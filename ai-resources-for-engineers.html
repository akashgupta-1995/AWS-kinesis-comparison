Inference & Serving

Getting your model to production efficiently. Quantization, KV cache, continuous batching, and the tools that make LLM serving economically viable.

Course Free
Efficiently Serving LLMs
DeepLearning.AI

Covers KV cache, continuous batching, quantization, speculative decoding, and how vLLM works. Exactly the production serving knowledge a senior engineer needs. Hands-on labs included. Free short course.

deeplearning.ai
Docs Free
vLLM Documentation
vLLM Project

The industry-standard LLM serving engine. Docs cover PagedAttention, continuous batching, quantization methods, multi-GPU serving, and deployment. If you're serving LLMs, start here.

docs.vllm.ai
Paper Free
Efficient Memory Management for LLM Serving with PagedAttention
Kwon et al., 2023 · arXiv

The paper behind vLLM's core innovation. Explains virtual memory paging applied to KV cache, why it achieves near-zero waste, and 20× throughput improvements. Short, readable, and highly important.

arxiv.org
Tool Free
llama.cpp
Georgi Gerganov · GitHub

C++ inference engine for running LLaMA-family models on CPU (and GPU). The reference implementation for local LLM inference and quantization. Understand it to understand INT4/INT8 quantization in practice.

github.com/ggerganov/llama.cpp
Post Free
The Transformer KV Cache Explained
Hugging Face Blog

Step-by-step explanation of KV caching mechanics, why it's essential for autoregressive decoding, memory implications, and how different serving frameworks manage it. A must-read before your first LLM deployment.

huggingface.co/blog
Docs Free
NVIDIA Triton Inference Server
NVIDIA

Production-grade model serving framework supporting PyTorch, ONNX, TensorRT, and more. Includes dynamic batching, model ensembles, multi-GPU, and observability. The enterprise-grade serving reference.

docs.nvidia.com
Also See: TGI — Text Generation Inference (HuggingFace) FlashAttention Paper Ollama (local LLM serving)
